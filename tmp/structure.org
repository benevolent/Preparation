items table
| String | Integer | String      |
|--------+---------+-------------|
| title  | level   | description |

実際に見えるルーティング
|------------------+--------+----------------------------------------------------|
| URL              | Action | 用途                                               |
|------------------+--------+----------------------------------------------------|
| /items           | index  | すべてのアイテムを一覧するページ                   |
| /items/hoge      | show   | title=hogeのアイテムを表示するページ               |
| /items/new       | new    | 新規アイテムを作成するページ(/itemsにまとめるかも) |
| /items/1/edit    | edit   | id=1のアイテムを編集するページ                     |
|------------------+--------+----------------------------------------------------|

RESTful routing
|--------------+---------------+---------+----------------------------------------------------|
| HTTP Request | URL           | Action  | 用途                                               |
|--------------+---------------+---------+----------------------------------------------------|
| GET          | /items        | index   | すべてのアイテムを一覧するページ                   |
| GET          | /items/1      | show    | id=1のアイテムを表示するページ                     |
| GET          | /items/new    | new     | 新規アイテムを作成するページ(/itemsにまとめるかも) |
| POST         | /items/create | create  | アイテムを作成するアクション                       |
| GET          | /items/1/edit | edit    | id=1のアイテムを編集するページ                     |
| PATCH        | /items/1      | update  | id=1のアイテムを更新するアクション(実際はPOST)     |
| DELETE       | /items/1      | destroy | id=1のアイテムを削除するアクション(実際はGET)      |
|--------------+---------------+---------+----------------------------------------------------|
